# Comment s’appelle cette association ?

> **Note sur la disponibilité des noms de domaine .fr** : pour chaque nom, la forme accolée (ex. `cheminscommuns.fr`) et, quand le nom comporte plusieurs mots, la forme avec tirets (ex. `chemins-communs.fr`) ont été testées par une simple requête HTTP HEAD (pas de whois, pour éviter le rate-limiting). ✅ = aucune réponse HTTP sur les deux formes (probablement libre, à confirmer par whois avant tout achat) ; ⚠️ = résultat mixte (une forme répond, l'autre non) ; ❌ = les deux formes répondent (très probablement prises). Le détail par domaine figure dans le tableau en fin de document.

# Une piste de réflexion

J'ai remarqué qu'en France, beaucoup d'associations de plaidoyer portent un nom qui décrit ce qu'elles défendent (Place au Vélo, Rue de l'Avenir, FUB, 60 Millions de Piétons). À l'inverse, les marques qui s'installent durablement expriment souvent le futur qu'elles souhaitent construire.

Dans cette logique, des noms comme :

* Chemins Communs  
* Ville Vivante  
* Place à Tous  
* Cap Commun

présentent plusieurs avantages. Ils portent une vision positive, sans enfermer l'association dans un mode de déplacement particulier. Ils permettent ainsi d'englober naturellement la marche, le vélo, l'accessibilité, les rues scolaires, le partage de l'espace public, ou plus largement toutes les mobilités actives et les politiques qui les favorisent.

D’aileurs, un simple mot peut être très efficace si jumelé avec une courte phrase, par exemple :

Association pour les mobilités actives du bassin nantais

En recherchant un équilibre entre mémorisation, inclusivité et potentiel de marque :

| Nom | .fr | Commentaire |
| :---- | :---- | :---- |
| Chemins Communs | ❌ | Sans doute mon préféré. Inclusif, facile à retenir, il évoque naturellement aussi bien la marche que le vélo. |
| Cap Commun | ❌ | Court, optimiste et porteur d'une dimension d'intérêt général. |
| Passerelles | ✅ | Belle métaphore des liens entre les lieux, les personnes et les quartiers. |
| Place à Tous | ✅ | Message fort sur le partage de l'espace public ; facile à mémoriser ; un peu trop proche à PAV. |
| Ville Vivante | ⚠️ | Positif et évocateur, même si le champ est un peu plus large que la seule mobilité. |
| Libres d'Aller | ✅ | Met l'accent sur les personnes et leur liberté de se déplacer plutôt que sur les infrastructures. |
| Voies Libres | ❌ | Dynamique et énergique, même si certains pourront y entendre une résonance ferroviaire (« voie libre »). |
| En Chemin | ⚠️ | Très simple, très mémorable et facilement appropriable ; bémol, ça fait penser à « En Marche ». |
